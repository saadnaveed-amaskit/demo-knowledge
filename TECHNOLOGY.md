# TECHNOLOGY

This file defines the required production technology stack for this repository.
When creating or modifying application code, AI agents should follow this stack unless explicitly instructed otherwise.

## Core Framework

- React 19+
- TypeScript
- Vite (project tooling and build)

## Styling and UI

- Tailwind CSS
- shadcn/ui components
- Framer Motion (animations and transitions)

## Routing and State

- React Router v7+ (routing)
- Zustand (UI state management)
- nuqs (URL state)

## Forms and Validation

- React Hook Form
- Zod

## Data, Tables, and Charts

- AG Grid (large data tables)
- Nivo (charts)

## Utilities

- date-fns

## Agent Guidance

- Prefer these libraries over introducing alternatives.
- If a needed capability is not covered here, call it out explicitly before adopting a new dependency.
